# Moteur de Recherche d'Information

Ce projet est un moteur de recherche simple développé en Python. Il permet de rechercher des informations dans un corpus de fichiers textes ou PDF.

## Technologies utilisées
- Python 3.x
- Streamlit
- scikit-learn
- TF-IDF et similarité cosinus

## Lancer le projet
```bash
pip install -r requirements.txt
streamlit run app/interface.py
